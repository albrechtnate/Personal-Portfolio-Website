Raw Type Alphabet Poster

Project Brief:

Find and document the 26 letters of the alphabet in the shapes of everyday objects or create the 26 letters out of physical objects. Create a “type as image” poster that displays all of these letters.

Project Solution:

I decided to create the 26 letters out of some of the different ingredients required to make chocolate chip cookies, as it would bring some diversity to the form and color of the letters while still retaining an overall unifying theme for the alphabet. Each letter was designed by hand using either chocolate chips, brown sugar, flour, white sugar, or an egg; photographed, and then edited in Photoshop. The letters and other graphical elements were later composited in InDesign to create the final poster.
