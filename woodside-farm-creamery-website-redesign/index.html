Woodside Farm Creamery Website Redesign

Project Brief:

Redesign the website for a business or organization, implement the website by building out the design using web technologies, and integrate the Wordpress Content Management System so that the business or organization owner can easily update the content themselves.

Project Solution:

I chose to redesign the website of Woodside Farm Creamery, a favorite ice cream shop near my hometown whose existing website design I felt did not reflect the high quality of their ice cream and facility. I began the process with taking notes about the existing content that would need to be transferred over and new content that would be added. I did lots of concept sketching to determine a visual solution that would make their site standout amongst the competition and communicate their existing “farmhouse” aesthetic.

Wireframes were created using Sketch, followed by design mockups created using Photoshop and Sketch. The site was developed using hand-crafted HTML and CSS and developed to look great across all different screen sizes.

The hard-coded HTML and CSS was then converted into a Wordpress theme. I created custom fields and widgets so that every bit of content on the site could be easily updated by the business owner if needed.

This project was created for educational purposes and is in no way affiliated with Woodside Farm Creamery.
