A Beginner’s Guide to Architecture Type Book

Project Brief:

Collect images of found type over the semester and create an A–Z book that incorporates ten images of each letter.

Project Solution:

I based the book on architecture styles that begin with each letter of the alphabet, and incorporated images of found type that matched each style. I wrote the content, collected and edited the images, designed the layout, and then printed and bound the book.
